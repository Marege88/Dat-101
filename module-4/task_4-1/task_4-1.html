1. [Why is object-oriented development so important in programming?]

    2. [Objektorientert utvikling er viktig fordi det gjør programmet lettere å strukturere, forstå og vedlikeholde. Ved å bruke objekter kan vi samle data og funksjoner som hører sammen i en enhet. Dette gjør koden lettere oversiktlig og den blir enklere og gjennbruke i større programmer.]

  2. [In Task 1, we create a static object called "AccountType." Explain what this object is used for and how this type of object can help us in software development.]

    3. [Det statiske objektet "AccountType" brukes til å lage faste verdier for ulike kontotyper. I stedet for å skrive tekst som "Brukskonto" eller "Sparekonto" flere steder i koden, kan vi hente verdier fra objektet. Dette gjør koden mer oversiktlig og reduserer risiko for feil. slike objekter er nyttige fordi de samler konstanter på et sted og gjør det lettere og endre programmet senere.]

  3. [Discuss why we use encapsulation in classes. Why do we want to "hide" code within our class? Explain how we can create "private" and "public" members in a constructor function.]

    4. [vi bruker innkapsling (encapsulation) for å beskytte data og gjøre koden mer sikker og oversiktelig. Ved å skjule interne variabler i en klasse kan vi hindre at de blir endret feil utenfra. Dette gjør programmet mer stabilt og lettere og kontrolere. Private medlemmer kan lages ved å bruke # forran variabel navnet i en klasse, for eksempel #balance. Public medlemmer er metoder og variabler som kan brukes utenfor klassen, som foreksempel deposit() eller withdraw(). i en constructor function kan vi lage private variabler ved å definere dem inne i funksjonen, og public metoder ved å bruke this.]

  4. [Explain what happens when we use the "new" command in JavaScript.]

    5. [Når vi bruker new i javaScript, opprettes et nytt objekt fra en klasse eller en constuctor. JavaScript lager et nytt objekt i minnet, kobler det til klassen og kjører constructoren. Deretter returneres det nye objektet slik at vi kan bruke det i programmet.]
